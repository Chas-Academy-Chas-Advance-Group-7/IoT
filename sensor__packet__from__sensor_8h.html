Detailed Description

Defines the data structure for sensor packets received from Arduino sensors.

This header declares the SensorPacket struct used for communication between sensors and the broker/backend system. The struct is packed to ensure there is no padding between fields, making it suitable for BLE or other binary transport protocols.

Example usage:

SensorPacket packet;
packet.sensor_id = 1;
packet.temperature = 23.5;
packet.humidity = 45.0;
packet.sensor_timestamp = millis();
