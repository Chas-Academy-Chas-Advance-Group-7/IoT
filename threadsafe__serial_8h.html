Detailed Description

Thread-safe Serial printing utilities for Arduino with FreeRTOS.

Provides safe wrapper functions around the Arduino Serial object that can be used across multiple FreeRTOS tasks. A global semaphore (serialMutex) is used to synchronize access and prevent concurrent writes to the Serial interface.

Example:

#include "utils/threadsafe_serial.h"
void setup() {
Serial.begin(115200);
serialMutex = xSemaphoreCreateMutex();
}
void loop() {
safePrintln("Hello from loop!");
vTaskDelay(pdMS_TO_TICKS(1000));
}
