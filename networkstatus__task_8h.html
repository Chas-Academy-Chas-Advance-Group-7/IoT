Detailed Description

Header for the network status FreeRTOS task.

Declares the networkStatusTask function which connects the ESP32 to Wi-Fi, monitors the connection status, and updates the networkEventGroup for other tasks to know when the network is available.

Example usage:

#include "tasks/networkstatus_task.h"
void setup() {
// Initialize FreeRTOS event group and mutex
// ...
// Create the network status task
xTaskCreate(networkStatusTask, "NetworkStatusTask", 4096, NULL, 1, NULL);
}

void networkStatusTask ( void *  pvParameters)

Task for monitoring Wi-Fi connection and updating network events.

Connects to Wi-Fi, sets or clears NETWORK_CONNECTED_BIT in the networkEventGroup, and continuously monitors the connection to attempt reconnection if Wi-Fi is lost.

Parameters
pvParametersUnused task parameter (required by FreeRTOS).
// Example of task creation
xTaskCreate(networkStatusTask, "NetworkStatusTask", 4096, NULL, 1, NULL);

Task for monitoring Wi-Fi connection and updating network events.

Connects to Wi-Fi using credentials from WiFi_secrets.h or fallback, waits for initial connection, and sets/clears NETWORK_CONNECTED_BIT in the networkEventGroup. Continuously monitors connection and attempts reconnection if Wi-Fi is lost.

pvParametersUnused task parameter (required by FreeRTOS).
